About

This page shows the last Dead by Daylight map my local OCR bot detected.

The bot reads the map name from the game, sends the latest result to a small Cloudflare Worker service, and this site reads that saved state. That means the page can keep showing the last detected map even when the bot is not running, and it only updates when my side sends a new map.

Project

Repo: dbd-clockmapdisplay

Current state

The current setup is mainly built around loading-screen OCR and tab/match-details OCR. It is still being worked on, but it already detects a lot of maps correctly.
